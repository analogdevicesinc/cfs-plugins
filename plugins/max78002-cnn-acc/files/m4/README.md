# MAX78002 CNN Image Classification Example

## Description

This project is an example that runs a cats-vs-dogs image classification model on the MAX78002.

## Software

### Project Usage

Universal instructions on building, flashing, and debugging this project can be found in the **[MSDK User Guide](https://analogdevicesinc.github.io/msdk/USERGUIDE/)**.

### Project-Specific Build Notes

(None - this project builds as a standard example)

## Required Connections

Connect a MAXPICO or other supported Debug adapter to the SWD Connector.
